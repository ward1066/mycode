# Title / description
1 [UC3] View Sales
Allows merchants to view a report on sales activity for products that have been purchased via the system. The merchant can choose different report types, view the report online, print it or save it locally in a different formats.
2 [UC8] Maintain Product Catalogue
The merchant wishes to update their product catalogue. They may wish to add, delete or update items within the catalogue. Updates might include changes to descriptions, prices (such as special offers) or a change to the delivery options that can be supported.
