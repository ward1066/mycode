Quality

95% of functional, non-functional tests must have passed unit, system and UAT test phases. All severity 1 and 2 defects will be resolved prior to live operations.

Operating Environment

ACME Online will be developed in Java and be deployed on a complaint J2EE appserver running on Linux. As this application is busines critical to ACME Plc. it will be deployed across two remote sites with both intra and inter-site redundancy.

The client environment cannot be managed or controlled. The application must run smoothly and reliably on all common combinations of OS and browser such that 98% of the target demographic are able to use the site reliably and securely.
