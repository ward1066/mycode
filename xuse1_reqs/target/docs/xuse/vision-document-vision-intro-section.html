Introduction

This document describes the vision for ACME Online. It provides a high-level summary of the expected features of ACME Online and defines the release roadmap within the context of the identified use-cases.

This document will endeavour to set the context for ACME Online; who the stakeholders are; the key business drivers; and the success criteria.

This document is expected to be a "living document": it will evolve with ACME Online as the project progresses. In this way the document should provide a good first reference for people new to ACME Online irrespective of the current progress on the project.

Document Audience

This document is indended for all stakeholders in ACME Online including ACME Plc., XML Solutions Ltd. business analysts, architects, developers, testers and service personnel.
